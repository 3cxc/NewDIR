# NewDIR


权限和称号简化给予系统 dirsystem


可在默认目录C:\DIRSave中找到数据文件


权限组可以自己定制例如(admin，owner)


设置为空的话就是都不给


称号颜色:red,blue,yellow之类的 [如果是rainbow则就是彩虹称号]
